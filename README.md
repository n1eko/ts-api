
# TS3 NODE API

A brief description of what this project does and who it's for


## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`TS3_HOST`

`TS3_SERVER_PORT`

`TS3_QUERY_PORT`

`TS3_USER`

`TS3_PASS`

`TS3_NICK`
